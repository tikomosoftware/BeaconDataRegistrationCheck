# Design Rules - Cool Modern

## Theme

クールでモダンなUIを採用する。

- ミニマル
- 情報整理重視
- 落ち着いた配色
- 余白を広く取る
- 少し未来感を持たせる
- 派手すぎない
- Apple / Linear / Vercel / Notion に近い空気感
